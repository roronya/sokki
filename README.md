# Sokki
Markup Language for memo.

## Concept

Sokki(速記)は、会議や講義のメモを取りながら、その場で情報を分類するためのマークアップ言語です。

メモを取っていると、話の記録・自分の補足・感想や疑問が入り混じって流れてきますが、書いている最中に整形している余裕はありません。Sokkiでは、すべてを上から下へ普通のテキストとして書き、行の分類は行末に記号を付けるだけで済ませます。

- 無印の行: 左カラム(主となる記録)
- 行末に ` >`: 中央カラム(補足・コメント)
- 行末に ` >>`: 右カラム(さらに一段メタな注記)
- 空行: セクションの区切り。HTMLでは同じ行に揃うので、記録とそれに対する注記が横に並ぶ

記号のスペースと `>` は全角でも書けます(`メモ　＞`)。日本語入力のままIMEを切り替えずに分類できます。行末をリテラルの ` >` で終わらせたいときは ` \>` のようにバックスラッシュでエスケープします。

記号を行頭ではなく行末に置くのがポイントで、行を書き終えた瞬間に付け足せるため、カーソルを戻す必要がなく筆記の流れを止めません。書くときは一次元(上から下)、読み返すときは二次元(3カラム)になる、というのが設計意図です。

### Example

```
今日の議題はリリース日程について
また延びるのでは >
9月に延期が決定
つらい >>
```

これをコンパイルすると、記録が左、補足が中央、感想が右に並んだ3カラムのHTMLが生成されます。

## Installation

```shell
$ go install github.com/roronya/sokki@latest
```

Make sure `$GOPATH/bin` (or `$HOME/go/bin`) is in your `$PATH`.

## Usage

```shell
$ sokki input.sk output.html
```

## Development

```shell
$ go test ./...
$ go build
```
